# Logic Gym

Logic Gym is a lightweight, browser-only logic puzzle app. Open the HTML file and start solving.

## How to run

1. Clone or download this repository.
2. Open `index.html` in your browser.

## Project structure

- `index.html` — The main page markup.
- `styles.css` — Styling for the layout and feedback states.
- `puzzles.js` — An array of available puzzles.
- `app.js` — UI wiring and answer checking.
